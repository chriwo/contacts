==============================================================
Feature: #47 - Add Distance Search for Contacts and Companies
==============================================================

See :issue:`47`

Description
===========

The plugin allows you to add a search plugin for addresses to a
page. The customer can enter a (german) zip address and a radius.
All contacts and companies within the specified radius are displayed.
